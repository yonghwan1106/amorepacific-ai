### Pain Point 2: 데이터 기반 의사결정 부족

| 현재 상태 | 문제점 |

### Pain Point 3: 경쟁 환경 고도화

- **인디 브랜드 급부상**: 민첩한 트렌드 대응으로 대기업 위협

- **글로벌 브랜드**: AI 맞춤형 기술 선점

- 국내: 그린워싱 문제로 제자리걸음
- 소비자층 세분화: 연령대별 선호도 차이 심화
  - 10~20대: 비건, 퍼스널컬러 중시
  - 30~50대: 리프팅, 탄력 중시

# 3. 솔루션 개요

## 3.1 BeautyTrend AI 정의

> **BeautyTrend AI**는 TikTok, Instagram, YouTube 등 소셜 미디어와 경쟁사 데이터를 실시간으로 수집·분석하여 **6~12개월 후의 뷰티 트렌드를 예측**하고, **자동화된 인사이트 리포트를 생성**하는 Multi-Agent AI 시스템입니다.

## 3.2 4대 핵심 역량

```
┌──────────────────────────────────────────────────────────────┐
│                    BeautyTrend AI                            │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌────────────┐  ┌────────────┐  ┌────────────┐  ┌────────┐ │
│  │  멀티소스   │  │  선행 예측  │  │  경쟁사    │  │ 자동   │ │
│  │  데이터    │  │  6~12개월  │  │  모니터링  │  │ 리포트 │ │
│  │  수집      │  │  트렌드    │  │            │  │ 생성   │ │
│  └────────────┘  └────────────┘  └────────────┘  └────────┘ │
│                                                              │
│   TikTok         성분/컬러/      신제품 출시    PDF/PPT      │
│   Instagram      텍스처 예측     가격/성분      자동 생성    │
│   YouTube                        분석                       │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### 역량 1: 멀티소스 데이터 수집

| 데이터 소스 | 수집 항목 | 갱신 주기 |
|-------------|-----------|-----------|
| TikTok | 해시태그, 조회수, 댓글 감정 | 실시간 |
| Instagram | 릴/포스트, 좋아요, 저장 수 | 실시간 |
| YouTube | 쇼츠, 시청시간, 트렌드 탭 | 일 1회 |
| 경쟁사 웹사이트 | 신제품, 성분, 가격 | 일 1회 |

### 역량 2: 6~12개월 선행 트렌드 예측

- **성분 트렌드**: 나이아신아마이드 → 다음 인기 성분은?
- **컬러 트렌드**: 팬톤 + 소셜 미디어 색상 통계 분석
- **텍스처 트렌드**: 젤, 크림, 세럼 등 제형 선호도 변화

### 역량 3: 경쟁사 신제품 모니터링

- 로레알, 에스티로더, 시세이도 등 글로벌 경쟁사
- 롬앤, 클리오 등 국내 인디 브랜드
- 중국 자국 브랜드 (퍼펙트 다이어리 등)

### 역량 4: 자동 트렌드 리포트 생성

- **주간 리포트**: 이번 주 급상승 키워드
- **월간 리포트**: 성분/컬러/텍스처 트렌드 종합
- **분기 리포트**: 6~12개월 예측 및 신제품 추천

## 4.2 각 에이전트 역할

### Orchestrator Agent (중앙 조정자)

| 역할 | 설명 |

| 기능 | 기술 |

| 감성 분석 | KoBERT, Sentiment Analysis |
| 키워드 추출 | NER, TF-IDF |

### Color Analysis Agent (컬러 분석)

| 기능 | 기술 |
|------|------|

| 시각화 | Matplotlib, Plotly |

| 기능 | 기술 |
|------|------|

# 5. AI 기술 상세

## 5.1 RAG (Retrieval-Augmented Generation)

### 아키텍처

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   Query     │     │  Retriever  │     │    LLM      │
│  (질의)     │ ──▶ │  (검색기)   │ ──▶ │ (생성기)    │
└─────────────┘     └─────────────┘     └─────────────┘
                          │
                          ▼
                   ┌─────────────┐
                   │ Vector DB   │
                   │ (Chroma/    │
                   │  Pinecone)  │
                   └─────────────┘
```

### 적용 방안

| 구성요소 | 기술 선택 | 이유 |
|----------|-----------|------|
| Embedding | OpenAI Ada-002 / KoSimCSE | 한국어 의미 유사도 우수 |
| Vector DB | Chroma (MVP) / Pinecone (Production) | 확장성 및 성능 |
| LLM | Claude 3.5 / GPT-4o | 최신 추론 능력 |
| Reranker | Cohere Rerank | 검색 결과 정확도 향상 |

### 시계열 필터링 + 벡터 검색 통합

```sql
-- TimeScale + pgvector 통합 쿼리 예시
SELECT content, embedding <=> query_embedding AS distance
FROM social_posts
WHERE created_at > NOW() - INTERVAL '3 months'
ORDER BY distance
LIMIT 10;
```

## 5.5 Self-Reflecting 기능

### 자가 개선 메커니즘

```
┌─────────────────────────────────────────────────────┐
│                Self-Reflecting Loop                 │
├─────────────────────────────────────────────────────┤
│                                                     │
│   ┌─────────┐   ┌─────────┐   ┌─────────┐         │
```
